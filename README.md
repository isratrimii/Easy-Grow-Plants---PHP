# Easy Grow Plants (PHP Version): AI-Powered Multi-Vendor Botanical Ecosystem

Easy Grow Plants is a comprehensive multi-vendor e-commerce platform designed to bridge the gap between plant enthusiasts and sellers.

## 🚀 Key Features

### 🛒 Multi-Vendor Marketplace
- **Three-Tier Architecture:** Dedicated portals for **Admin**, **Seller**, and **Customer** to ensure streamlined operations and data integrity.
- **Comprehensive Catalogue:** Browsing and purchasing systems for Plants (Indoor, Outdoor, Medicinal), Fertilizers, and Planters.
- **Advanced Shopping Workflow:** Includes a persistent Shopping Cart, Wishlist, and a secure Checkout process.
- **Order Management:** Real-time tracking of orders from placement to delivery status.

### 🧠 Smart AI & Automation
- **AI Virtual Assistant:** A 24/7 intelligent Chatbot designed to assist users with care guides, species information, and navigation.
- **Care Library:** Automated repositories for plant-specific care instructions, including watering schedules and sunlight requirements.

### 🛡️ Administrative & Seller Tools
- **Inventory Lifecycle:** Full CRUD (Create, Read, Update, Delete) operations for sellers to manage products and stock levels.
- **User Verification:** Admin oversight for seller authentication and product moderation.
- **Financial Security:** Integrated payment gateways (Bkash, Nagad, etc.) with data encryption for safe transactions.

---
## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (Core)
* **Database:** MySQL
* **Tools:** XAMPP/WampServer


## 🚀 How to Run Locally
1. Clone the repository to your local machine.
2. Move the project folder to the `htdocs` directory of your XAMPP installation.
3. Import the `.sql` database file using **phpMyAdmin**.
4. Configure the database connection in your PHP files.
5. Access the project via `localhost/Easy-Grow-Plants---PHP`.
